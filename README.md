# help-seller
고객명과 개인통관고유부호가 일치하는 지 확인할 수 있는 기능, CJ대한통운, 우체국 택배의 배송조회를 할 수 있는 사이트입니다.
리엑트를 사용하여 제작하였으며 Rest api, express를 사용했습니다.

[help-seller](https://help-seller.herokuapp.com)

## 설치하기
```shell
npm i 
```

## 로컬에서 띄우기
```shell
npm run client
```

## 빌드 및 배포
```shell
git push heroku master
```
